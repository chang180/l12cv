## 🔧 技術規格

### 系統需求
- PHP 8.4+
- Composer
- Node.js 18+
- SQLite (開發) / MySQL (生產)

### 主要技術
- Laravel 13.x
- Livewire 4.x
- Tailwind CSS 4.x
- Flux UI
- Laravel Socialite
- Alpine.js

### 開發工具
- Pest (測試)
- Laravel Pint (程式碼風格)
- Laravel Sail (Docker)
- Vite (前端建構)
